# Other Components

## Overview

Learn more about the supporting tools that extend Gardener's capabilities:

- [**Dependency Watchdog**](https://github.com/gardener/dependency-watchdog/blob/master/README.md): Monitors critical service health and handles disruption recovery
- [**etcd-druid**](https://github.com/gardener/etcd-druid/blob/master/README.md): Automated etcd cluster lifecycle management and operations
- [**Gardener Discovery Server**](https://github.com/gardener/gardener-discovery-server/blob/main/README.md): Provides public metadata for Gardener resources
- [**Machine Controller Manager**](https://github.com/gardener/machine-controller-manager/blob/master/README.md): Declaratively manage machines for your Kubernetes clusters
