# Gardener Extension Provider metal-stack

The extension for the metal-stack provider enables Gardener to manage shoot clusters on [metal-stack](https://metal-stack.io/) infrastructure.

> [!NOTE]
> The metal-stack extension is maintained by metal-stack in their own GitHub organization.

You can find more information in the [`metal-stack/gardener-extension-provider-metal`](https://github.com/metal-stack/gardener-extension-provider-metal) GitHub repository.
