# Other Components

## Overview

Learn more about the supporting tools that extend Gardener's capabilities:

- [**Dependency Watchdog**](/docs/other-components/dependency-watchdog/): Monitors critical service health and handles disruption recovery
- [**etcd-druid**](/docs/other-components/etcd-druid/): Automated etcd cluster lifecycle management and operations
- [**Gardener Discovery Server**](/docs/other-components/gardener-discovery-server/): Provides public metadata for Gardener resources
- [**Machine Controller Manager**](/docs/other-components/machine-controller-manager/): Declaratively manage machines for your Kubernetes clusters
